**Phase 0: Foundations of Computing and Project Introduction**
===========================================================

**Introduction**
---------------

Welcome to Phase 0 of the Personal Computer Museum project! In this phase, we will lay the groundwork for understanding the basics of computing and introduce the project that you will be working on throughout the learning roadmap. The Personal Computer Museum is an interactive website that showcases the evolution of computers, and you will be playing a crucial role in building it.

**Project Context**
-----------------

* **Project:** Personal Computer Museum
* **Tagline:** Explore the History of Computing
* **Description:** The Personal Computer Museum is an interactive website that showcases the evolution of computers, from the first binary systems to modern operating systems.
* **Stack:** HTML, CSS, JavaScript

**Phase Objectives**
-------------------

By the end of Phase 0, you will be able to:

* Understand the basic components of a computer and its functions
* Learn about binary and number systems
* Understand bits, bytes, and data representation
* Familiarize yourself with operating system basics, file systems, and command line interfaces
* Understand how the internet works and set up a development environment
* Introduce yourself to programming concepts and growth mindset for engineering

**Topic-to-Project Connection Map**
-----------------------------------

The following topics will be covered in Phase 0, and you will apply your learning to the Personal Computer Museum project:

| Topic | Project Connection |
| --- | --- |
| 0.1 What is a Computer? | Add a brief introduction to what a computer is and its basic components. |
| 0.2 Binary & Number Systems | Create a binary calculator to demonstrate binary and number systems. |
| 0.3 Bits, Bytes & Data Representation | Design a data representation system to showcase bits, bytes, and data types. |
| 0.4 Operating System Basics | Build a simple operating system simulator to demonstrate basic OS concepts. |
| 0.5 File Systems & Directories | Create a virtual file system to organize and manage files and directories. |
| 0.6 Command Line & Terminal | Implement a command line interface to interact with the virtual file system. |
| 0.7 How the Internet Works | Design a simple network diagram to illustrate how the internet works. |
| 0.8 Dev Environment Setup | Set up a development environment to build and host the museum website. |
| 0.9 What is Programming? | Write interactive coding challenges to teach programming concepts. |
| 0.10 Growth Mindset for Engineering | Reflect on your learning journey and write a growth mindset statement for engineering. |

**What to Expect**
------------------

In Phase 0, you will:

* Learn the fundamentals of computing and computer science
* Apply your learning to real-world problems and projects
* Develop a growth mindset for engineering and problem-solving
* Set up a development environment to build and host the Personal Computer Museum website

**Getting Started**
-------------------

To get started with Phase 0, review the topics and project connections outlined above. Make sure you have a basic understanding of HTML, CSS, and JavaScript, as these will be the primary technologies used for the project.
