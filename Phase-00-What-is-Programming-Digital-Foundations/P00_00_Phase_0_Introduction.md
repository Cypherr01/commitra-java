**Phase 0 Introduction: Foundations of Computing and Development Environment**
====================================================================

### Phase Overview

Welcome to Phase 0 of the Personal Computer Museum project, where you will lay the foundation for your journey in creating an interactive website that explores the history of computing. In this phase, you will delve into the basics of computing, including what constitutes a computer, binary and number systems, data representation, operating system basics, file systems, command line interfaces, and networking fundamentals. Additionally, you will set up your development environment and introduce yourself to programming concepts, all while adopting a growth mindset essential for engineering and lifelong learning.

### Phase Objectives

By the end of Phase 0, you will be able to:
- Explain the basic components and functions of a computer.
- Understand and apply binary and number systems in computing contexts.
- Design simple systems to demonstrate data representation and operating system basics.
- Create a virtual file system and interact with it using a command line interface.
- Describe how the internet works and its importance in modern computing.
- Set up a development environment suitable for web development.
- Introduce programming concepts through interactive challenges.
- Reflect on your learning journey and articulate a growth mindset for engineering.

### Phase Topics and Project Connection

The following topics will be covered in Phase 0, along with their connection to the Personal Computer Museum project:

| Topic | Project Connection |
| --- | --- |
| 0.1 What is a Computer? | Learners add a brief introduction to what a computer is and its basic components. |
| 0.2 Binary & Number Systems | Learners create a binary calculator to demonstrate binary and number systems. |
| 0.3 Bits, Bytes & Data Representation | Learners design a data representation system to showcase bits, bytes, and data types. |
| 0.4 Operating System Basics | Learners build a simple operating system simulator to demonstrate basic OS concepts. |
| 0.5 File Systems & Directories | Learners create a virtual file system to organize and manage files and directories. |
| 0.6 Command Line & Terminal | Learners implement a command line interface to interact with the virtual file system. |
| 0.7 How the Internet Works | Learners design a simple network diagram to illustrate how the internet works. |
| 0.8 Dev Environment Setup | Learners set up a development environment to build and host their museum website. |
| 0.9 What is Programming? | Learners write interactive coding challenges to teach programming concepts. |
| 0.10 Growth Mindset for Engineering | Learners reflect on their learning journey and write a growth mindset statement for engineering. |

### Phase Deliverables

- A binary calculator demonstrating binary and number systems.
- A data representation system showcasing bits, bytes, and data types.
- A simple operating system simulator.
- A virtual file system with a command line interface.
- A simple network diagram illustrating how the internet works.
- A set up development environment for web development.
- Interactive coding challenges teaching programming concepts.
- A growth mindset statement for engineering.

### Assessment and Evaluation

Your progress and understanding in Phase 0 will be assessed through the completion of the phase topics and the quality of the deliverables. Continuous learning and the application of concepts to the Personal Computer Museum project are key to successfully completing this phase.

### Conclusion

Phase 0 of the Personal Computer Museum project is designed to provide you with a comprehensive introduction to the foundations of computing and set you up for success in your web development journey. By completing the topics and deliverables outlined in this phase, you will not only gain a deeper understanding of computing fundamentals but also begin to build the skills necessary to create an interactive and educational website that explores the history of computing.
